###分支的管理
当前所有分支的列表
	$ git branch
在列表中在当前所在的分支签有 `*`。

查看各个分支最后一个提交对象的信息。

	$ git branch -v
从该清单中筛选出你已经（或尚未）与当前分支合并的分支，可以用 `--merged` 和 `--no-merged` 选项。

与当前分支合并的分支：

	$ git branch --merged
如果在列表中没有 `*` 的分支可以用 `git branch -d` 来删掉。

尚未与当前分支合并的分支：

	$ git branch --no-merged
这个列表的分支，不能用 `git branch -d` 删除该分支，会提示错误，因为那样做会丢失数据。

但是如果你确定要删除该分支上的改动，可以用大写的删除选项 `-D` 强制删除。
